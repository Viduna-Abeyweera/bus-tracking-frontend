# 🚍 Real-Time Bus Tracking System — Frontend

A React application for real-time bus tracking with interactive maps.

## Tech Stack
- React 18
- Vite
- Leaflet + OpenStreetMap
- React Router DOM
- React-Leaflet

## Features
- 🗺️ Interactive map showing live bus locations
- 🚌 Custom bus markers with labels
- 🚏 Bus stop markers on map
- 📍 Driver GPS sharing with start/stop
- 🕐 ETA calculation to selected bus stop
- ⏱️ Real-time updates every 3 seconds
- 📱 Responsive design

## Pages
- **Home** — Landing page with role selection
- **Passenger View** — Map + ETA + bus stop selector
- **Driver Dashboard** — GPS sharing with live stats

## Setup
1. Install Node.js 18+
2. Run: `npm install`
3. Run: `npm run dev`
4. Opens at `http://localhost:5173`
5. Make sure backend is running at `http://localhost:8080`
